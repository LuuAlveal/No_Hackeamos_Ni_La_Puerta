Aplicacion completamente espumante 🍾🍾 para Anotarse a Materias previas

GUIA DE INSTALACION
1. Instalar dependencias
    ``` bash
    npm install
    ```
2. Instalar alertas
   ```bash
   npm install sweetalert2
   ```
3. Iniciar proyecto
   ```bash
   npm start
   ```
Apartir de abrir esta app digale "El Hechicero", porque esta a punto de ver magia.
