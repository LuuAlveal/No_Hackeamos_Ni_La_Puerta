Aplicacion completamente espumante 🍾🍾 para Anotarse a Materias previas

GUIA DE INSTALACION
--------DENTRO DEL PROYECTO------------
1. Instalar dependencias
    ``` bash
    npm install
    ```
2. Instalar alertas
   ```bash
   npm install sweetalert2
   ```
3. Iniciar proyecto
   ```bash
   npm start
   ```
